# 视频播放器

标签（空格分隔）： python

---
## 测试环境

python 3.6

win10

## 代码功能

在输入框中输入自己想要观看的视频链接，编辑播放，即可观看。


## 使用的库

`tkinter、webrowser`

我们以电影《机器之血》为例：

我们首先复制视频链接：

## 代码运行方法

将全部文件[下载](https://github.com/lhx1228/Video_Player)下来，运行`Video_Player.py`即可。
